# Contain-CLI

**Contain-CLI** — легковесный оркестратор контейнеров на Python для локальной разработки и небольших проектов. Запускает многоконтейнерные приложения через YAML-конфиги с авто-перезапуском, health checks, Prometheus-метриками и веб-дашбордом.
---

##  Возможности

- **Запуск сервисов** — поднимает все контейнеры одной командой из YAML-конфига
- **Авто-перезапуск** — watchdog мониторит контейнеры и перезапускает упавшие
- **Health checks** — HTTP и команды проверки здоровья для каждого сервиса
- **Метрики** — Prometheus exporter на порту 9092
- **Веб-дашборд** — Flask-дашборд со статусом контейнеров и графиками
- **Управление** — просмотр логов, exec в контейнеры, массовый stop/start
- **Сборка образов** — автоматический билд из Dockerfile при старте
- **YAML-конфигурация** — простой и понятный формат
